# bililive-vibration-player

播放包含 bilibili 直播震动数据的视频，并通过连接的手柄同步震动反馈。

支持播放 `bililive-vibration-parser` 预先生成的 `.json` 文件，或者选择本地文件在线解析。

## 开发

```bash
pnpm install
pnpm --filter='bililive-vibration-player' run dev
```
