# bililive-vibration

哔哩哔哩直播震动数据解析器 & 播放器。

在线 Demo：[vib.meowers.me](https://vib.meowers.me)

## 项目结构

本仓库为 Monorepo，包含以下子项目：

- `bililive-vibration-parser`：震动数据解析器，用于从视频文件中解析出震动数据。
- `bililive-vibration-player`：震动数据播放器，用于随视频播放解析出的震动数据。

每个子项目的文档请查看对应目录下的 README。
